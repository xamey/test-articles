Visite chez les pompiers

Calandreta Albi février 3, 2013
Les calandrons sont allés visiter la caserne des pompiers à Albi.
Il sont revenus avec pleins d'images dans la tête. Voilà leurs travaux:


 Les pompiers entrent dans la caserne.

Les pompiers vont éteindre le feu d'une maison.

Les pompiers vont chercher un homme qui s'est brûlé avec de l'eau bouillante dans sa maison.

Les pompiers arrosent l'immeuble incendié.
Les pompiers arrivent avec l'ambulance pour sortir le monsieur de la voiture accidentée.

Les pompiers viennent sauver un bébé animal qui était perdu dans la haute montagne.

Les pompiers sauvent un chat qui était monté sur un arbre.

Les pompiers arrivent avec le camion pour éteindre le feu d'une maison.


Merci aux sapeurs-pompiers du centre départemental d'incendie et de secours du Tarn pour leur accueil.
